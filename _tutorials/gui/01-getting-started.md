<div class="note">Tip: You can follow this tutorial in a file or using the REPL</div>

Creating a new GUI

```lua
local myGui = gui.new()
```


Opening it
```lua
myGui.open()
```

![basic gui](/assets/img/gui-tutorial/gui-1.png)

Closing it
```lua
myGui.close()
```
